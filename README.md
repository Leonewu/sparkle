## 目标

1. 同时支持按需引入/后编译
2. 单元测试
3. travis ci
4. 全局样式，不用 scoped，module，方便外部覆盖
5. 版本管理 semver 规范
6. 兼容性 Android4.0+， IOS8+
7. 使用 rem，类名遵循 BEM
8. eslint，babel， commit lint
9. 制定文档模板，文档生成 vuepress?
10. ts，vue3.0?

1. 框架搭建，vue-cli，配置 eslint，babel，webpack，创建文件目录，输出 npm 包
2. 成功引入，并实现按需加载或者后编译
3. 跑通测试用例
4. 生成一份文档
